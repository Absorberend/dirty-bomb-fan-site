Dirty Bomb config files

I get a lot of questions about what settings I use and if people can have my config. You can download my config below and get the same settings as I do.

Dirty Bomb is heavily tweak-able. From HUD elements to visuals, almost everything can be changed to some degree. Dirty Bomb uses four files to save your settings, which are these:

  • ShooterEngine.ini (Graphics/Base settings)
  • ShooterGame.ini (Game settings)
  • ShooterInput.ini (Binds)
  • ShooterUI.ini (HUD)

These files can be found at:

*:\My Documents\My Games\UnrealEngine3\ShooterGame\Config
* being whatever hard drive your \My Documents folder is located at.

You can download my files (settings) here:

  •  Config download

If you need help installing the config see the instructions below.

Config installation

  1.  Go to \Documents\My Games\UnrealEngine3\ShooterGame\Config
  2.  Select ShooterEngine, ShooterGame, ShooterInput & ShooterUI
  3.  Copy them to a random folder to your desktop for backup
  4.  Copy the .ini files (from the download below) into the Step 1 folder and overwrite them
  5.  Right click each following file and select properties to see if they are read only. ShooterEngine, ShooterInput and ShooterUI should all be read only.
  6.  Run the game, check on how it runs.

Startup options

If you're interested I have some startup options as well to remove bloom and fog from the game. This really helps and also bootst your FPS. All you have to do is create a file, let's call it exec.txt, in \SteamApps\common\Dirty Bomb\Binaries. In that file write every command on a new line, like this:

  1. BloomSize 0
  2. FogDensity 0
  3. ColorGrading 2
  4. RawInputUpdateTest true
  5. Stat FPS

Then rightclick Dirty Bomb in Steam, select Set Launch Options in the General tab and add -exec=exec.txt (screenshot)

If you want the .txt file somewhere else you can also use a full path after -exec, just make sure you put it in quotes if the path contains spaces.
